## Missing Markers Priority

When generating the "what to order next" list, prioritize in this order:
1. CRP (most clinically actionable — drives SSRI augmentation decisions)
2. Ferritin + Transferrin saturation (HFE monitoring — safety critical)
3. Homocysteine (methylation assessment)
4. 25(OH) Vitamin D (VDR variants)
5. HLA-B27 serology (one-time, confirms/rules out 20x AS risk)
6. ESR (baseline inflammation)
7. Fecal calprotectin (gut-joint axis)
8. B12 / Folate (methylation)

## German Lab Format Notes

Common German lab report formats:
- **MDI Limbach**: Tabular format with Analyse/Resultat/Einheit/Referenz-/Zielbereich columns
- **Synlab**: Similar tabular, may include graphical reference bars
- **Bioscientia**: PDF with marker/value/unit/reference columns
- **Labor Berlin**: Mixed format
